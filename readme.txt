Программа через API бирж получает данные о курсах пар криптовалют.
Анализирует их и выводит на экран такие, которые выгодно купить и перепродать с указанием бирж, на которых это можно сделать.

Главный файл Program.cs

Абстрактный класс BasicCryptoMarket, от него наследуются классы для соответствующих описания бирж: PoloniexMarket, BittrexMarket, 
ExmoMarket, YobitMarket. В классах есть методы для запросов к api (public void LoadPairs(string command)). Вся информация о загруженных парах криптовалют хранится в списках, 
элементами которого являются экзэмпляры класса ExchangePair - описание каждой пары криптовалют (Цена покупки, продажи, название биржи).

Но в программе используются только первые три биржи, так как у последней неудобный API, приходиться делать много запросов, что сильно 
влияет на быстродействие (Проверить работу с использованием всех 4х бирж можно, раскомментировав // YobitMarket yobit = new YobitMarket();
в Program.cs).

Класс PairsListOperator анализирует списки пар криптовалют каждой биржи и записывает в свой список те пары, которые выгодно купить и 
перепродать с помощью метода AnalysePairs(BasicCryptoMarket[] marketsArray).
В этом же методе определяются кроссы-курсы, которые также записываются в список List<ExchangePair> _actualPairs этого класса.

В результирующем выводе идентификатором [WARN] помечаются пары, анализ которых может быть ошибочным (либо слишком большое различие в
курсах на разных биржах), а идентификатор [CROSS] служит для определения кросс-курсов.
